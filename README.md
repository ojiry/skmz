Skmz is a Rails engine that shows the schemas.

## Installation

Add this line to your application's Gemfile:

```ruby
group :development do
  gem 'skmz'
end
```

And then execute:

```bash
$ bundle
```

## Usage

Visit `/skmz/schemas` in your app.
